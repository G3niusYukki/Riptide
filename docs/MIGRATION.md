# 从 Clash Verge Rev 迁移

Riptide 完全兼容 Clash YAML 配置格式，迁移非常简单。

## 配置迁移

### 1. 导出配置

在 Clash Verge Rev 中：
1. 打开"配置"页面
2. 点击"导出"按钮
3. 保存 `.yaml` 文件

### 2. 导入到 Riptide

1. 打开 Riptide
2. 点击"导入配置"
3. 选择导出的 `.yaml` 文件
4. 完成！

## 订阅迁移

订阅 URL 格式完全相同，直接复制粘贴即可。

## 常见问题

**Q: 可以共用配置文件吗？**
A: 可以，配置格式完全相同。

**Q: 订阅会冲突吗？**
A: 不会，可以同时在两个客户端使用相同订阅。

**Q: 如何同步配置？**
A: Riptide v1.1+ 支持 WebDAV 配置同步，可以跨设备同步。
