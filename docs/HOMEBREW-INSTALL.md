# Homebrew 安装说明

## 添加 Tap

```bash
brew tap G3niusYukki/riptide
```

## 安装 Riptide

```bash
brew install riptide
```

## 更新 Riptide

```bash
brew update
brew upgrade riptide
```

## 卸载

```bash
brew uninstall riptide
brew untap G3niusYukki/riptide
```

## 故障排除

如果安装失败，尝试：
```bash
brew doctor
brew cleanup
brew install --verbose riptide
```
